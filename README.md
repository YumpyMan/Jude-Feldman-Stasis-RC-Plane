# Jude-Feldman-Stasis-RC-Plane


**Description**

My project is a completely scratch-built RC trainer plane. It has 2 main parts: the actual plane with receiver, and the controller with the transmitter. The controller has 3 inputs from the human pilot: a left stick for steering with rudder-aileron mixing, a right stick for roll and pitch control, and a slider for throttle adjustment. The transmitter uses an nRF24L01 radio module to send values to the receiver in the plane, which then tells the aileron, rudder, elevator, and ESC what to do to control the plane. All physical pieces of the plane will be printed out of LW-PLA or PLA-Aero in chunks, which will then be glued together. 

**Why I Made This Project**

I have always thought things that fly are super cool, and recently I've become super obsessed with aerospace engineering, rocketry, and model flying. However, nice RC planes are usually a few hundred bucks, and I wanted to try and make one myself. I also saw this as an opportunity to improve my maker skills, specifically CAD, programming, and electronics. Mainly, I just wanted to build something cool and this seemed like an awesome project.
